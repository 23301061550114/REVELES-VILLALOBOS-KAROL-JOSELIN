# REVELES-VILLALOBOS-KAROL-JOSELIN
La Tienda Demo es una simulación de una tienda en línea donde los usuarios pueden ver productos por categorías, consultar su información y agregarlos al carrito. El proyecto demuestra el uso de HTML, CSS y JavaScript para crear una plataforma básica de compras. También incluye una función extra para mejorar la experiencia del usuario.
